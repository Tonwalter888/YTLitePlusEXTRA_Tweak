# YTweaks

Adds "Fullscreen to the Right" option for the iOS YouTube app.

More settings and tweaks are planned. Designed to work with a [fork of YTLite](https://github.com/fosterbarnes/YTLite) (also known as [YouTube Plus](https://github.com/dayanch96/YTLite)).

## Supported YouTube versions

Confirmed version 20.20.7 and newer. Lower are either untested or unsupported.
